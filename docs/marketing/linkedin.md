## Auto-mod proofing — read before posting

- **Put the link in the first comment for the two longer posts** (Founder story, Shorter update) — LinkedIn's algorithm measurably suppresses reach on posts with an external link in the body; it's a ranking penalty, not a removal, but it behaves like soft moderation. The short Feature-focused post can keep an inline link since it's secondary to the main text.
- **No engagement-bait phrasing** ("comment YES below," "tag someone who needs this," "repost if you agree") — LinkedIn explicitly down-ranks and can flag posts using these patterns.
- **Cap hashtags at 3–5, only on-topic ones** (e.g. #buildinpublic, #indiehackers, #SaaS) — hashtag stacking reads as spam to both the algorithm and human reviewers.
- **Don't publish near-identical text as multiple posts close together** — repost/duplicate detection applies here too, same as other platforms.
