## Auto-mod proofing — read before posting

- **Link goes in the first comment, not the post body**, for every group post below. Facebook's spam classifier and most group admin-approval queues hold or shadow-limit posts that contain an outbound link directly in the body — a link-free post with the link in the first comment routes around most of that.
- **Check the group's rules for a self-promo day/flair first.** Many indie hacker / SaaS founder groups only allow product posts on a designated day or with a specific post type selected — posting outside that window is the most common reason for an instant admin removal, not the copy itself.
- **Don't post identical text to multiple groups back-to-back.** Facebook's spam detection looks at repeated near-identical content across groups/pages from the same account in a short window, same as Reddit's — vary the opening line and CTA per group.
- **Skip hashtag stacking and multiple exclamation points.** Groups/Pages content gets down-ranked by the same signals used to detect low-quality/promotional spam; one hashtag max, and only if the group culture actually uses them.
- **Don't use urgency language** ("act now," "limited time," "don't miss out") — combined with an external link, that pattern is a core signal in Facebook's commercial-content/spam classifier even in organic (non-ad) posts.
