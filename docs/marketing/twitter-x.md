Format: single tweets + one thread. Short, direct, no emoji spam, no fabricated numbers.

## Auto-mod proofing — read before posting

- **Put the link in a reply to your own tweet, not the tweet itself.** X's ranking algorithm suppresses reach on posts with an outbound link in the body — it's not removal, but it functions like a soft auto-mod. A link-free tweet + self-reply with the link gets both the reach and the click.
- **Never post the same tweet text twice.** X's spam detection flags (and can temp-lock) accounts posting duplicate or near-duplicate content repeatedly — each standalone tweet below is written distinctly for this reason; don't reuse one across multiple days verbatim.
- **Zero or one hashtag, only if directly relevant.** Hashtag stacking is a spam-classifier signal, not a discovery boost, on X.
- **Don't mass-reply the same template to many accounts in a short window.** That pattern (identical text, many recipients, short time) is exactly what X's spam/abuse detection is built to catch — the reply template below should be hand-tailored per conversation, not copy-pasted repeatedly.
- **New/low-follower accounts posting links get less reach and closer spam scrutiny** — build a short posting history without links before leading with a launch tweet.
