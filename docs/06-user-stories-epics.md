# 06. User Stories & Epics

> Only one worked example exists so far. Break every remaining feature (Goal analysis, channel selection, plan generation, todo execution, etc.) into stories using the same format.

## Epic: Campaign Creation

### Story

As a founder,
I want to enter my business goal,
so AI can recommend the best marketing channels.

**Acceptance Criteria**

- Given I enter a goal
- When I submit
- Then AI validates the goal
- And recommends channels.
