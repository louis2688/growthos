# 08. User Flows

## Campaign Creation Flow

```
User
 │
 ▼
Enter Goal
 │
 ▼
AI analyzes goal
 │
 ▼
AI researches channels
 │
 ▼
Show recommendations + reasons
 │
 ▼
User selects channels
 │
 ▼
AI creates:
    Campaign
    Goal
    Plans (1 per selected channel)
    Todos for each plan
 │
 ▼
Campaign Dashboard
```

### Worked example

**Goal:** Get my AI budgeting app to 100 users.

**AI recommends:**

- r/personalfinance
- Product Hunt
- Indie Hackers
- Facebook Budgeting Group

**User selects:**

- r/personalfinance
- Product Hunt
- Indie Hackers

**AI creates:**

- Campaign: Launch AI Budgeting App
- Goal: 100 Signups in 30 Days
- Plans: Reddit Outreach, Product Hunt Launch, Indie Hackers Promotion

**Todos — Reddit Outreach**

- [ ] Read subreddit rules
- [ ] Generate educational post
- [ ] Create supporting graphic
- [ ] Publish
- [ ] Reply to comments
- [ ] Measure CTR

**Todos — Product Hunt**

- [ ] Create gallery
- [ ] Write tagline
- [ ] Prepare FAQs
- [ ] Launch
- [ ] Engage with comments

## Dashboard → Execution Flow

```
Dashboard
 ↓
Create Campaign
 ↓
Enter Goal
 ↓
AI Analysis
 ↓
Channel Selection
 ↓
Generate Campaign
 ↓
Campaign Dashboard
 ↓
Execute Todos
```
